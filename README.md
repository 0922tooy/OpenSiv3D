# OpenSiv3D #

## Platforms ##

  * Windows Desktop
  * Mac OS X
  * Universal Windows Platform (Not supported yet)

## Requirements ##

#### Windows Requirements ###

  * Windows 7 SP1 / 8.1 / 10
  * Microsoft Visual C++ 2015 Update 2 or newer
  
#### Mac OS X Requirements ###

  * Mac OS X v10.11 El Capitan or newer
  * Xcode 7.3 or newer
